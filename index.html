¿Qué es CodExpress?

CodExpress es una competencia de programación del capítulo estudiantil Computer Society UNAL. Su propósito es desafiar las habilidades de los estudiantes y mostrar cómo los aprendizajes en matemáticas y física se relacionan con la computación.

Niveles de competencia

Nivel Básico

Fundamentos y lógica de programación

Simuladores matemáticos simples y lógica en Python.

Relación con cursos: Fundamentos de matemáticas, Programación I.

Nivel Intermedio

Estructuras de datos y POO inicial

Proyectos en Python, Java o C++.

Ejemplos: sistema de colas, agendas académicas.

Nivel Avanzado Ligero

Algoritmos y eficiencia

Algoritmos de búsqueda (BFS, DFS), optimización en Python y C++.

Relación con cursos: Algoritmos, Matemáticas discretas.

Nivel Integración y Sistemas

Aplicaciones completas

Apps con bases de datos y APIs (Python + JavaScript).

Ejemplo: sistema CRUD, chat simple, integración de servicios.

Recursos de preparación

Material recomendado para fortalecer tus habilidades antes del evento.

▶ Playlist de fundamentos en Python (UNAL)

Refuerza lógica, sintaxis y resolución de problemas en Python.

 ▶ Programming for Everybody – Univ. of Michigan

Curso introductorio para dominar programación desde cero.

 ▶ Data Structures and Algorithms – UC San Diego

Domina estructuras, algoritmos y análisis de complejidad.

 ▶ Bases de datos en SQL con Python – IBM

Aprende a conectar Python con bases de datos relacionales.
